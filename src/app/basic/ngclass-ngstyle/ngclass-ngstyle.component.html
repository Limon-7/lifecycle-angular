NgStyle: NgStyle gives you fine grained control on individual properties. But if you want to make changes to multiple properties at once, creating a class which bundles those properties and adding the class with NgClass makes more sense.

  • {{ person.name + " from " + person.country }}

Ng Class

The NgClass directive allows you to set the CSS class dynamically for a DOM element.

  • {{ person.name + " from " + person.country }}

Class with function

We can also set a class on an element by binding to the input property binding called class
